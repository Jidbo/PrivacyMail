Scanwords

Scanwords are used to scan an email for subscribe-links. These potential subscribe links are then programatically clicked to regiser a new newsletter. A Name Scanword is used to scan the clickable name of a link. A link scanword is used to scan the actual url of the link. If a link contains either one of these words, the link is registered to be clicked. Blacklist words are used to blacklist a Link. If a blacklist word is in a name or url of a link, the link is automatically discarded, even if the link contains a name or link scanword.
